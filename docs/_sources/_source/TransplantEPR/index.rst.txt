Transplant EPR Project
======================

Official project designation currently being written as part of the Go Digital initiative. Basic aim (presently) is to
incorporate Transplant procedures and dataflow into the OUH EPR (Cerner Millienium) systems.

About the Project
-----------------

AKA: was Project OTTR, now Project Emory (as of Jun 2017)

Two existing documents are available: A collection of notes from one of the department's surgeons, and the other
is a requirements gathering draft outlining workflows for Transplant patients.
